MarkDown语法

发表于2022-02-06更新于2022-02-06

块级:
标题
#标题1

##标题2

###标题3

####标题4

#####标题5

######标题6

引用:引用一段话  前面有个竖杠
>这是一段引用

列表:
    有序:使用数字
把大象放进冰箱:
    1.把大象放进冰箱
    2.把大象塞进去
    3.关上冰箱

    无序:使用短横线/星号
    - 芜湖1
    - 芜湖2
    * 芜湖3

任务列表: - []  输入x就会勾选上
    明天要做的事:
    - [] 吃饭
    - [] 睡觉
    - [x] 打豆豆

代码块: 三个 ``` 加上编程语言
    ``` php

数学公式:四个美元符号  中间把公式括起来
    $$
        数学公式
    $$ 

表格:
    表头:
        |姓名|年龄|成绩
    对齐方式:
        |:--- 左对齐
        |---: 右对齐
        |:--:| 居中对齐

脚注:
    脚注的内容:
        名字[^name]
            [^name]:Whoami

        横线:
            ---
          
    脚注的链接: [名字](链接 "这里的类似于a标签的title")
            [百度](http://www.baidu.com "一个搜索引擎")

         引用链接:在文章中出现多次的情况
            [百度][id]
                 [id]:baidu.com "一个搜索引擎"

         请参考[标题1](#标题1)  这里括号的名字要和标题1一样
             

URL:
     http://www.baidu.com

图片:
     ![百度](https://www.baidu.com/img/flexible/logo/pc/result.png "百度logo")

行内:
斜体
加粗
console.log("使用一对``反引号编写行内代码");
下划线: 需要使用u标签
下划线
表情: :加上表情的名称:
:smile:
H2O X^2^
对文字进行高亮:
==这是一段高亮文字==
